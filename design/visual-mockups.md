```
Card Size: 63mm x 88mm (Standard TCG)
Bleed Area: +3mm all sides
Safe Zone: -3mm from edges
Text Size: Minimum 8pt for readability
Line Weight: Minimum 0.5pt for printing
```

### Digital Optimization
```
Screen Sizes:
Mobile: 375x667px min
Tablet: 768x1024px
Desktop: 1920x1080px+

Performance Targets:
60fps animation capability
<500kb per card asset
<100ms interaction response
Battery-conscious rendering
```
